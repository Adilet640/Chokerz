# Инструкция по настройке инфоблоков в админке 1С-Битрикс (продолжение)

### 2. Страницы (Инфоблок типа "Информация")

**Символьный код:** `pages`

#### Шаг 1: Создание инфоблока "Страницы"

1. Войдите в админку 1С-Битрикс
2. Перейдите в раздел **"Контент"** → **"Инфоблоки"**
3. Нажмите **"Добавить инфоблок"**
4. Заполните поля:
   - **Тип инфоблока:** Выберите или создайте тип "Информация"
   - **Название:** Страницы сайта
   - **Символьный код:** `pages`
   - **Сайт:** Выберите сайт (например, `s1`)
   - **Группы пользователей:** Выберите группы (например, "Все пользователи")
   - **Активность:** Отмечено
5. Нажмите **"Сохранить"**
