# Инструкция по настройке интеграций для интернет-магазина CHOKERZ

## 2. Интеграция оплаты

### 2.1. ЮKassa

#### Шаг 1: Получение учётных данных ЮКасса (упрощённо)

1. Зарегистрируйтесь на сайте [ЮКасса](https://yookassa.ru/)
2. В личном кабинете получите:
   - **Shop ID** (идентификатор магазина)
   - **Секретный ключ** (для подписи запросов)
   - **Тестовый режим** (да/нет)
   - **Валюта** (RUB)
   - **Способы оплаты** (карты, СБП, электронные кошельки и т.д.)

#### Шаг 2: Настройка в 1С-Битрикс

1. Войдите в админку 1С-Битрикс
2. Перейдите в раздел **"Магазин"** → **"Настройки магазина"** → **"Платёжные системы"**
3. Нажмите **"Добавить платёжную систему"**
4. Выберите **"ЮКасса"** из списка доступных платёжных систем (или установите модуль ЮКасса, если его нет)
5. Заполните поля:
   - **Название:** ЮКасса (онлайн-оплата)
   - **Активность:** Отмечено (галочка)
   - **Shop ID:** Вставьте ваш идентификатор магазина (например, `123456`)
   - **Секретный ключ:** Вставьте ваш секретный ключ (например, `test_1234567890abcdef`)
   - **Тестовый режим:** Выберите (да/нет)
   - **Валюта:** Выберите (RUB)
   - **Способы оплаты:** Выберите (карты, СБП, электронные кошельки и т.д.)
   - **Описание:** Вставьте описание (например, "Оплата картой, СБП, электронными кошельками через ЮКассу")
   - **Сортировка:** Укажите сортировку (например, 100)
   - **Группы пользователей:** Выберите группы (например, "Все пользователи")
   - **Страны:** Выберите страны (например, "Россия")
   - **Регионы:** Выберите регионы (например, "Все регионы")
   - **Города:** Выберите города (например, "Все города")
   - **Почтовые индексы:** Укажите почтовые индексы (например, "Все индексы")
   - **Минимальная сумма заказа:** Укажите минимальную сумму (например, 0)
   - **Максимальная сумма заказа:** Укажите максимальную сумму (например, 1000000)
