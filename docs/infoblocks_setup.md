# Инструкция по настройке инфоблоков в админке 1С-Битрикс

## Общая информация

Данный документ содержит пошаговую инструкцию по созданию и настройке инфоблоков для интернет-магазина "CHOKERZ — амуниция для животных".

## Требования

- Доступ к админке 1С-Битрикс (Бизнес редакция)
- Права администратора на создание и редактирование инфоблоков

## Инструкция по созданию инфоблоков

### 1. Каталог товаров (Инфоблок типа "Каталог")

**Символьный код:** `catalog_products`

#### Шаг 1: Создание инфоблока каталога товаров (тип "Каталог")

1. Войдите в админку 1С-Битрикс
2. Перейдите в раздел **"Магазин"** → **"Торговые каталоги"**
3. Нажмите **"Добавить каталог"**
4. Заполните поля:
   - **Название:** Каталог товаров CHOKERZ
   - **Символьный код:** `catalog_products`
   - **Сайт:** Выберите сайт (например, `s1`)
   - **Группы пользователей:** Выберите группы (например, "Все пользователи")
   - **Активность:** Отмечено
5. Нажмите **"Сохранить"**

#### Шаг 2: Настройка свойств инфоблока каталога товаров

1. Перейдите в раздел **"Магазин"** → **"Торговые каталоги"** → **"Каталог товаров CHOKERZ"**
2. Нажмите **"Свойства"**
3. Добавьте следующие свойства:
   
   **a) Тип изделия (список)**
   - Название: `Тип изделия`
   - Символьный код: `TYPE`
   - Тип свойства: `Список`
   - Значения:
     - Ошейник (символьный код: `ошейник`)
     - Поводок (символьный код: `поводок`)
     - Намордник (символьный код: `намордник`)
     - Комплект (символьный код: `комплект`)
   - Отображать в детальном просмотре: Отмечено
   - Отображать в списке: Отмечено

   **b) Материал (список)**

   - Символьный код: `MATERIAL`
   - Тип свойства: `Список`
   - Значения:
     - Кожа (символьный код: `кожа`)
     - Нейлон (символьный код: `нейлон`)
     - Металл (символьный код: `металл`)
     - Текстиль (символьный код: `текстиль`)

   - Отображать в списке: Отмечено
   
   **c) Назначение (список)**

   - Символьный код: `PURPOSE`
   - Тип свойства: `Список`
   - Значения:
     - Для собак (символьный код: `для_собак`)
     - Для кошек (символьный код: `для_кошек`)
     - Универсальное (символьный код: `универсальное`)
   - Отображать в детальном просмотре: Отмечено
   - Отображать в списке: Отмечено
   
   **d) Размер (строка)**

   **e) Цвет (строка)**

#### Шаг 3: Настройка торговых предложений (SKU)

1. Перейдите в раздел **"Магазин"** → **"Торговые каталоги"** → **"Каталог товаров CHOKERZ"**
2. Нажмите **"Торговые предложения"**
3. Включите торговое предложение (галочка)
4. Настройте свойства для торговых предложений:
   - Размер (строка)
   - Цвет (строка)
   - Материал (строка)
5. Нажмите **"Сохранить"**
