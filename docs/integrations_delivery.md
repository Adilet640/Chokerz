# Инструкция по настройке интеграций для интернет-магазина CHOKERZ

## Общая информация

Данный документ содержит пошаговую инструкцию по настройке интеграций для интернет-магазина "CHOKERZ — амуниция для животных".

## Требования

- Доступ к админке 1С-Битрикс (Бизнес редакция)
- Права администратора на настройку модулей и интеграций
- API ключи для интеграций (СДЭК, Почта России, ЮKassa, Robokassa)
