# Страницы сайта (шаблоны для создания через компонент "Создание страницы" в админке)

## Описание:
Данные файлы являются шаблонами для создания страниц через компонент "Создание страницы" в админке 1С-Битрикс. Они содержат примеры подключения компонентов для различных страниц сайта.

## Страницы:

### 1. Каталог товаров (`catalog.php`)
- Компонент: `bitrix:catalog`
- SEF-режим: включён
- URL: `/catalog/`

### 2. Детальный просмотр товара (`product.php`)
- Компонент: `bitrix:catalog.element`
- SEF-режим: включён (наследуется от каталога)
- URL: `/catalog/#SECTION_CODE#/#ELEMENT_CODE#/`

### 3. Блог (`blog.php`)
- Компонент: `bitrix:news`
- SEF-режим: включён
- URL: `/blog/`

### 4. Контакты (`contacts.php`)
- Компонент: `bitrix:main.include`
- URL: `/contacts/`

## Инструкция по использованию:

1. Войдите в админку 1С-Битрикс
2. Перейдите в раздел "Контент" → "Страницы"
3. Нажмите "Добавить страницу"
4. Выберите "Создать страницу на основе шаблона"
5. Выберите нужный шаблон из списка (или скопируйте код из файла в редактор)
6. Настройте параметры компонента (например, `IBLOCK_ID`)
7. Сохраните страницу
