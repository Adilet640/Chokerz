# Шаблон сайта CHOKERZ

Адаптивный шаблон интернет-магазина для амуниции животных на 1С-Битрикс.

## Структура

```
chokerz/
├── .description.php        # Описание шаблона
├── header.php              # Шапка сайта
├── footer.php              # Подвал сайта
├── template_styles.php     # Подключение стилей
├── template_styles.css     # Базовые стили
├── styles/                 # CSS стили
│   ├── main.css           # Основной файл стилей
│   ├── base/              # Базовые стили
│   ├── blocks/            # Блоки по BEM
│   └── layout/            # Макет
├── js/                    # JavaScript
│   ├── main.js           # Точка входа
│   ├── modules/          # Модули
│   └── vendor/           # Библиотеки
└── images/               # Изображения
    ├── logo.svg          # Логотип
    ├── favicon.ico       # Иконка сайта
    └── og-image.jpg      # Open Graph изображение
```

## Особенности

- Адаптивная верстка (Desktop, Tablet, Mobile)
- Ванильный JavaScript (ES6+)
- BEM-нейминг для классов
- Поддержка современных браузеров
- Оптимизация производительности

## Настройка

1. Установить шаблон через панель администрирования Битрикс
2. Настроить свойства страниц (SEO мета-теги)
3. Подключить шрифты и иконки
4. Настроить изображения (логотип, фавикон)

## Технические требования

- 1С-Битрикс: Управление сайтом — Бизнес редакция
- Поддержка PHP 7.4+
- Поддержка MySQL 5.7+
