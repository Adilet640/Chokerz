# Компонент избранного (кастомный)

wishlist — кастомный компонент избранного для интернет-магазина "CHOKERZ — амуниция для животных".

## Параметры компонента:
- CACHE_TIME (int) — время кэширования (по умолчанию 3600)
## Пример использования:

```php
<?php
$APPLICATION->IncludeComponent(
    "custom:wishlist",
    ".default",
    Array(
        "CACHE_TIME" => "3600"
    )
);
?>
```
