## Описание проекта
Интернет-магазин «CHOKERZ — амуниция для животных» на CMS 1С-Битрикс (Бизнес редакция).

**Цель:** Разработка адаптивного интернет-магазина с каталогом товаров, корзиной, личным кабинетом, блогом и интеграцией с маркетплейсами (Озон, ВБ, ЯМ).

**Основные требования:**
- Адаптивная верстка (Desktop, Tablet, Mobile)
- Ванильный JavaScript (ES6+), BEM-нейминг
- Инфоблоки: Каталог товаров, Страницы, Блог, Отзывы, Видео, Преимущества
- Фильтрация и сортировка товаров
- Корзина и оформление заказа (sale.order.ajax)
- Интеграция доставки: СДЭК, Почта России, Озон
- Интеграция оплаты: ЮKassa, Robokassa
- Личный кабинет с профилем, заказами, избранное (wishlist)
- Интеграция отзывов с Озон и ВБ (API + парсинг)
- Мета-теги, каноникалы, пагинация для SEO
- Производительность: кэширование, WebP, lazy load

**Дизайн:** Макеты в Figma — https://www.figma.com/design/Oua0ebXLGqQTooiRhskMH8/CHOKERZ?node-id=0-1&p=f&t=T9hiztjIZfNYIIIz-0

**Хостинг:** Разработка ведётся на сервере по адресу https://dev.chokerz.ru/
